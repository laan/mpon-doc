小游戏分享
======

分享页url为：
<http://aa.yz057.com/game_share.html>

###传递参数 `get`


| 参数名		|		说明		| 是否必须	|
| ---- 		| ----				| ----		|
| schema		| 					| 是 		| 
| game_id		| 游戏id 			| 是 		|
| app_id 		| 渠道appID 		| 是		|
| download	| app下载地址 		| 否		| 


###示例

<http://aa.yz057.com/game_share.html?schema=platform&game_id=4&app_id=194857>
